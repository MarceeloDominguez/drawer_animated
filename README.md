## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/drawer_animated.git
```

Go to the project directory

```bash
  cd drawer_animated
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
